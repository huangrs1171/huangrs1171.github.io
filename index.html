如何高效实用Git

一、node安装

curl –silent –location https://rpm.nodesource.com/setup_8.x | bash -

yum install -y nodejs

node -v

v8.15.0

npm -v

6.4.1
NPM即Node包管理器（Node Package Manager），它是一个以JavaScript脚本语言编写的软件包管理系统，默认环境为Node.js，使用NPM可以在环境部署时避免手动引入各种功能包，可以提升包管理的效率。
使用淘宝NPM镜像替代官方版本，使用淘宝定制的CNPM命令工具替代官方的NPM：

npm install -g cnpm –registry=https://registry.npm.taobao.org

二、安装Hexo框架（Hexo是一个快速、简洁且高效的JavaScript博客框架），使用Markdown（或其他渲染引擎）解析文章。

cnpm install -g hexo

三、在本地初始化博客

hexo init myblog
